Nebari

Nebari

Nebari is an open-source data platform that automates the deployment of scalable and cost-effective data science environments. It integrates JupyterHub, Cloud environments, and Dask Gateway to streamline workflows for both HPC and Kubernetes setups.

With minimal DevOps overhead, Nebari enables teams to focus on data science tasks while maintaining robust and flexible infrastructure.

Adversarial Robustness Toolbox

Adversarial Robustness Toolbox

The Adversarial Robustness Toolbox (ART) is an open-source Python library designed to assist developers and researchers in building robust and trustworthy machine learning models. It provides tools for defending against adversarial attacks, testing model vulnerabilities, and improving robustness.

Armory

Armory

Armory is a testbed designed to run scalable evaluations of adversarial defenses in machine learning. It facilitates reproducible research by providing configurable Docker environments for testing models under adversarial conditions. Armory integrates with the Adversarial Robustness Toolbox (ART), allowing flexible combinations of attacks and defenses.

Note: This repository, GARD-Armory, is specific to the DARPA GARD research program. For the actively maintained version, refer to Armory Library.

DataEval

DataEval

DataEval focuses on characterizing image data and its impact on model performance across classification and object-detection tasks.

Data Analytics

The scope of data analytics is very broad, but as DataEval is developed, we look for impactful metrics as guides towards feature development.

  • Model-agnostic metrics: relevance/completeness/coverage, metafeatures (data complexity)
  • Model-specific metrics: dataset sufficiency, data/model complexity mismatch
  • Post-deployment monitoring metrics: dataset-shift metrics, model performance bounds under covariate shift

These metrics guide model error assessment, retraining strategies, and provide bounds on model performance under changing data conditions.

Explainable AI Toolkit (XAITK)

XAITK

The Explainable AI Toolkit (XAITK) contains a variety of tools and resources to help users, developers, and researchers understand complex machine learning models. The toolkit combines a searchable repository of independent contributions and a more integrated, common software framework.

The toolkit was developed under the Defense Advanced Research Projects Agency (DARPA) Explainable Artificial Intelligence (XAI) program.

Natural Robustness Toolkit (NRTK)

NRTK

The nrtk package is an open-source toolkit for evaluating the natural robustness of computer vision algorithms to various perturbations, including sensor-specific changes to camera focal length, aperture diameter, and more.

Functionality is provided through Strategy and Adapter patterns to allow for modular integration into systems and applications.

The nrtk-jatic package supports AI T&E use-cases and workflows, through interoperability with the maite library and integration with other JATIC tools. Users seeking to use NRTK to perturb MAITE-wrapped datasets or evaluate MAITE-wrapped models should start with the nrtk-jatic package.

JATIC Toolbox (MAITE)

MAITE (Modular AI Trustworthy Engineering)

MAITE is a library of common types, protocols, and utilities designed for the test and evaluation (T&E) of supervised machine learning models. It supports seamless workflows across JATIC projects by providing essential components for synergistic integration.

Developed under the JATIC program, MAITE eliminates redundancies and reduces the burden of implementing shared functionalities in machine learning T&E. The library is lightweight, frequently updated, and ensures compatibility with other JATIC tools.
